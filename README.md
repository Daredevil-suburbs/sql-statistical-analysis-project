# SQL Statistical Analysis Project

This project demonstrates how SQL can be used for statistical analysis of customer transactions.

## Techniques Used

- Descriptive Statistics
- Percentile Analysis
- Outlier Detection (IQR)
- Correlation Analysis
- Frequency Distribution
- Z Score Analysis

## Dataset

Synthetic transaction dataset including:
- transaction_id
- customer_age
- amount
- days_since_signup
